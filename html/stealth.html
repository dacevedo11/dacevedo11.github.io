UE5 Stealth Game Framework

Technical Designer / Gameplay Programmer

Platform: PC
Engine: Unreal Engine 5
Team Size: 1
Duration: 3 months

Tools Used:
  • Unreal Engine 5
  • Blueprints
  • Metahumans
  • Megascans
Description:

A short demo for a stealth game made in Unreal Engine 5 using Blueprints. Mantle over obstacles and take down enemies while avoiding detection to steal the Jewel and escape.

Key Features and Technical Implementation:
Stealth Mechanics
  • Dynamic line of sight detection system
  • Cover-based movement and concealment
  • Interaction system for takedowns and traversal
Enemy AI
  • Behavioral state machines with patrol and chase states
  • Blackboard system for AI data management and decision making
  • Patrol behavior with waypoints using Unreal's navigation mesh
  • Detection mechanics with sight cones
Combat Animation
  • Fluid takedown animations
  • Motion warping for seamless character positioning and targeting
  • Ragdoll physics integration for realistic interactions
  • Mantle system using ray tracing for height detection
Environment
  • Multi-tiered alarm system
  • Dynamic lighting
  • Level blockout and layout design
