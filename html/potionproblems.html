Potion Problems

2D Online Multiplayer Party Game
// 1. Description

Potion Problems is a 2D online multiplayer party game. It is a Thesis/Advanced Game Project directed by 3rd year MFA student, Alison Seidner. I listed this first because my experience on this project has inspired and motivated me to lead a thesis project of my own. My role is as a volunteer Usability Researcher. My responsibilities include conducting playtests, analyzing data, and providing feedback to the team to improve the game's user experience.
