Retro Robot Rumble

Combat Designer / Technical Designer

Platform: PC
Engine: Unity
Team Size: 20+
Duration: In development

Tools Used:
  • Unity
  • C#
  • GitHub
  • ClickUp
  • Figma
Description:

Retro Robot Rumble is a 3D top-down hack-n-slash roguelike where players build their own robot, with each part determining their abilities and playstyle. Set in an elementary school, players battle rivals and aim to defeat the rich bully kid in a weekly tournament.

My Responsibilities:
  • Designed combat systems for an upcoming 3D top-down hack-n-slash roguelike game, including player melee mechanics, dodge/dash systems, and modular robot part abilities
  • Created enemy design framework encompassing common and elite enemy types, attack patterns, abilities, traits, and AI behavior trees for combat encounters
  • Collaborated with engineering and art teams to implement combat mechanics, enemy behaviors, and robot customization systems
  • Developed major game systems including robot part slide decks, macro sheets, and pathing for elite enemies in arena-based combat
