Best Friends Forever

2D Platformer and AI Interactive Experience
// 1. Description

Best Friends Forever is a 2D platformer and AI interactive experience. You play as Cassie, a tech-savvy and music loving 16 year-old girl, who is dealing with the loss of her best friend, Jamie. In order to cope with Jamie’s passing, Cassie trains an AI chatbot to talk just like Jamie. This game uses the OpenAI API so that the player can chat directly with ChatGPT in the game. This game was made for a joint-final project for CTIN 534 and 541 at USC. I was the Lead Programmer, and my responsibilities included coding, level design, UI design, and implementation of art assets.

As the game progresses, Cassie discovers items that bring back memories of her friendship with Jamie, gradually uncovering her deep sense of guilt over the accident that caused Jamie’s death. With the AI’s guidance, Cassie embarks on a journey of self-forgiveness, ultimately learning to move forward in her grieving process. This game uses the OpenAI API so that the player can chat directly with ChatGPT in the game. ***Due to the game's requirements, an API key must be stored in a local root folder for it to function, so a working build is currently unavailable to share.
